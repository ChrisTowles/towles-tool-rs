# Brainstorms & Prototypes

Before wiring anything up, brainstorm and prototype so I can react. If it's
visual, make a single self-contained HTML file with several wildly different
directions using fake data. If it's an approach, search the codebase and give me a
ranked list of options from cheapest to most ambitious. I'll tell you which
resonates.
