# Blind Spot Pass

Do a blind spot pass on what I'm about to work on: find my relevant unknown
unknowns — what I don't know to ask about, what good looks like, the potholes to
avoid — and explain them to me so I can prompt you better. Ask me what I already
know first if that would sharpen the pass.
